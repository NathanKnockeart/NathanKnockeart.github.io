What are the benefits of version control? Version Control is a way to keep track of all the changes you made. You can also revisit any save point that you like. For example if want to recall my second save point I could with version control.

How does git help you keep track of changes? Git helps you keep track of changes by making commits. Also there is untracked files and files that need to be commited. Once commited they will save on your computer. If you would like to check your commmit and the time you made them, type in the command "git log".

Why use GitHub to store your code? One reason why you should use GitHub is everything is free. Another, is that git is easily linked with GitHub. You can use git to edit/create your files and use GitHub to share those files with others.
